Usage

QCERT-cockpit can be entered through the transaction /MXP/QC_QCERT.

Selection Screen​

When entering the cockpit the selection can be influenced through parameters restricting the selectedd status as well as select-options restricting on the attributes of the desired certificate entries.

Operation​

Once the selection is made, the worklist with all desired certification entries can be seen and worked through.

Buttons and Functions​

ButtonDescriptionRemark
ReleaseMoves the selectedd certificate entry into the next release status, according to the status profile
Cancel Release Moves the selectedd certificate entry into the previous release status, according to the status profile
Cancel CoA output Cancels the output of an already released certificate entry, resetting its status to INIT. Additionally existing DMS-Entries for the certificate will be locked
 Edit BlockBlocks the certificate entry for further processing
 Undo BlockUnblocks the certificate entry for further processing
Preview Calls the preview function of the provided printing program, or BAdI
Status management Provides a better overview about the status of the certificate entry, including change documents
Display application log Shows, if available, the application log for the selected certificate entry
Navigation Profile Opens the SAP-Navigation Profile Menu to implement custom functions to QCERT-cockpit

Hyperlinks​

In order to get more detailed information about the certificate entry, most table content referencing to other documents are hyperlinks which bring the user directly to the display transaction for the document. e. g. sales document, delivery or batch
